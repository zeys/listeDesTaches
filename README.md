# listeDesTaches
Outil de gestion de tâches (To-Do list) réalisé avec Laravel


![alt text](https://i.imgur.com/ynQAQ1H.jpg)

git clone https://github.com/zeys/listeDesTaches.git todolist  
cd todolist  
configurer le fichier /.env 
  
DB_CONNECTION=mysql  
DB_HOST=127.0.0.1  
DB_PORT=3306  
DB_DATABASE=todo  
DB_USERNAME=root  
DB_PASSWORD=  
  
composer install  
php artisan migrate
